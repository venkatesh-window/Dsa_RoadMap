Binary Trees: Traversals and Problems

1. Binary Tree Traversals

Depth-First Search (DFS) Traversals

Inorder Traversal (Left-Root-Right)

Algorithm:

  1. Traverse the left subtree
  2. Visit the root node
  3. Traverse the right subtree

Use Cases: Returns nodes in non-decreasing order for BSTs

Preorder Traversal (Root-Left-Right)

Algorithm:

  1. Visit the root node
  2. Traverse the left subtree
  3. Traverse the right subtree

Use Cases: Copying trees, serialization

Postorder Traversal (Left-Right-Root)

Algorithm:

  1. Traverse the left subtree
  2. Traverse the right subtree
  3. Visit the root node

Use Cases: Deleting trees, expression tree evaluation

Breadth-First Search (BFS) Traversal

Level Order Traversal

Algorithm:

  1. Use a queue to process nodes level by level
  2. Visit nodes at each level from left to right

Use Cases: Finding depth, level-specific operations

3. Implementation Examples

Recursive Inorder Traversal

void inorder(TreeNode root) {
    if (root == null) return;
    inorder(root.left);
    System.out.print(root.val + " ");
    inorder(root.right);
}

Iterative Level Order Traversal

void levelOrder(TreeNode root) {
    Queue queue = new LinkedList<>();
    queue.add(root);
    while (!queue.isEmpty()) {
        TreeNode current = queue.poll();
        System.out.print(current.val + " ");
        if (current.left != null) queue.add(current.left);
        if (current.right != null) queue.add(current.right);
    }
}

4. Key Concepts

  • Tree Height: Maximum depth from root to leaf
  • Balanced Tree: Height difference between subtrees ≤ 1
  • Complete Binary Tree: All levels filled except possibly last, nodes left-aligned
  • Full Binary Tree: Every node has 0 or 2 children
  • Binary Search Tree: Left subtree nodes ≤ root ≤ right subtree nodes
