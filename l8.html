Bit Manipulation

Introduction

Bit manipulation is the act of algorithmically manipulating bits or other pieces of data shorter than a byte. It involves direct control of bits at the hardware level, making it extremely efficient for certain operations. This technique is fundamental in low-level programming, cryptography, and optimization problems.

Why Learn Bit Manipulation?

  • Direct hardware control leads to faster execution
  • More memory-efficient operations
  • Essential for systems programming and embedded systems
  • Common in technical interviews and competitive programming

Binary Number System

Computers store all data in binary format using bits (0s and 1s). Understanding binary representation is crucial for bit manipulation.

Binary Basics

  • Each bit represents a power of 2
  • Rightmost bit is the least significant bit (LSB)
  • Leftmost bit is the most significant bit (MSB)
  • Numbers can be represented in different bases: decimal, binary, hexadecimal

Bitwise Operators

AND (&)

Returns 1 only if both bits are 1

Example: 5 & 3 = 1 (101 & 011 = 001)

OR (|)

Returns 1 if at least one bit is 1

Example: 5 | 3 = 7 (101 | 011 = 111)

XOR (^)

Returns 1 only if bits are different

Example: 5 ^ 3 = 6 (101 ^ 011 = 110)

NOT (~)

Flips all bits (1's complement)

Example: ~5 = -6 (in 32-bit system)

Left Shift (<<)

Shifts bits to the left, filling with 0s

Example: 5 << 1 = 10 (101 becomes 1010)

Right Shift (>>)

Shifts bits to the right

Example: 5 >> 1 = 2 (101 becomes 10)

Important Bit Manipulation Concepts

Setting a Bit

Use OR operation with a mask: num | (1 << position)

Clearing a Bit

Use AND operation with complement: num & ~(1 << position)

Toggling a Bit

Use XOR operation: num ^ (1 << position)

Checking if Bit is Set

Use AND operation: (num & (1 << position)) != 0

Counting Set Bits

Use Brian Kernighan's algorithm or built-in functions

Power of Two Check

A number is power of two if (n & (n-1)) == 0 and n > 0

Common Bit Manipulation Techniques

Masking

Using bit masks to extract specific bits

Bit Mask Operations

  • Create mask: mask = (1 << n) - 1
  • Extract bits: value & mask
  • Set bits: value | mask
  • Clear bits: value & ~mask

Swapping Numbers

Using XOR to swap without temporary variable:

a = a ^ b

b = a ^ b

a = a ^ b

Finding Unique Element

Using XOR to find element that appears odd number of times
